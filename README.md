## What is this? / 这是什么？

**EN:** Bar1 is a GenAI experience center built by AI. It showcases how to build, deploy, and operate GenAI applications using OpenClaw + AWS. Every line of code in this repo was written by AI assistants (Kiro + OpenClaw).

## Built with OpenClaw / 由 OpenClaw 驱动

This project is powered by [OpenClaw](https://github.com/openclaw/openclaw), a personal AI agent platform running on AWS.

AWS AI Portfolio

从芯片到应用，从模型到智能体——AWS AI 全栈产品图谱，一页看懂。

🛸Frontier Agents — 前沿智能体
新一代自主 AI Agent，能独立工作数小时甚至数天，无需持续人工干预。代表 AI 从"工具"到"队友"的进化。
Kiro Autonomous Agent NEW
你的虚拟开发者。能维持跨仓库的深度上下文，学习团队的编码风格，独立完成复杂的开发任务——从 bug 修复到功能开发，可以连续工作数天。
场景：复杂 bug 修复、跨仓库重构、功能开发、技术债清理
同类：Devin, OpenAI Codex Agent, Claude Code
AWS DevOps Agent NEW
自主运维 Agent，持续监控应用健康状态，自动检测异常、诊断根因、执行修复。将被动告警响应转变为主动问题预防。
场景：7×24 自动化运维、故障自愈、性能优化、变更管理
同类：PagerDuty AIOps, Datadog AI Agent
AWS Security Agent NEW
自主安全测试 Agent，能理解应用架构和源代码，自动执行渗透测试、发现漏洞、验证风险。将定期安全测试转变为持续安全验证。
场景：持续渗透测试、漏洞扫描、安全合规验证、代码安全审查
同类：Snyk AI, Wiz AI Agent
🚀Applications & Agents — 应用与智能体
开箱即用的 AI 应用和工具，面向开发者、业务用户和企业，直接解决实际问题。
Kiro IDE
AI 驱动的开发环境，基于 Code OSS。内置 Spec 驱动开发、Steering 文件、MCP 集成、自定义 Agent。支持 Vibe Coding 和结构化开发两种模式。
场景：日常编码、代码审查、架构设计、项目脚手架生成
同类：Cursor, Windsurf, GitHub Copilot Workspace
Amazon Quick Suite
从 QuickSight 进化而来的 AI 驱动商业智能平台。集成 AI Agent 进行数据分析、深度研究、自动化报告，用自然语言查询数据。
场景：业务数据分析、管理层仪表盘、自动化报告生成
同类：Power BI + Copilot, Tableau + Einstein
AWS Transform
AI 驱动的应用现代化服务。部署专业 AI Agent 自动完成代码分析、重构、语言翻译、框架迁移，将项目时间缩短 80%+。
场景：Java 版本升级、.NET 到 Linux 迁移、单体应用拆分微服务
同类：Google Gemini Code Assist (migration), 无直接竞品
Amazon Connect
AI 驱动的云联络中心，集成自然语言理解、实时语音分析、智能路由、Agent 辅助。
场景：客服中心智能化、语音 IVR、实时通话分析和质检
同类：Genesys Cloud, Five9, Twilio Flex
AWS Marketplace（AI）
AI 模型和应用的市场，提供数百个预训练模型、SaaS 应用和数据集，一键部署到你的 AWS 环境。
场景：快速获取第三方 AI 模型、评估不同供应商方案
同类：Azure Marketplace, Hugging Face Hub
🧠AI & Agent Development — AI 与智能体开发
构建 AI 应用和智能体的核心平台。从基础模型到 Agent 框架，从知识库到安全护栏，一站式开发。
AMAZON BEDROCK · 基础模型平台
Amazon Nova 自研模型
AWS 自研基础模型家族，包括 Nova Micro（文本）、Nova Lite（多模态）、Nova Pro（平衡）、Nova Premier（最强）。Nova Sonic 支持实时语音对话，Nova Canvas 支持图像生成。
场景：企业级 AI 应用，多模态理解，实时语音助手，图像生成
同类：GPT-4o, Claude, Gemini
第三方模型（3P Models）
通过 Bedrock 统一 API 访问 Anthropic Claude、Meta Llama、Mistral、Cohere、Stability AI 等顶级模型，无需管理基础设施。
场景：根据任务选择最优模型，A/B 测试不同模型效果，避免供应商锁定
同类：Azure OpenAI Service, Google Model Garden
模型定制（Customization）
支持 Fine-tuning 和 Continued Pre-training，用企业私有数据定制基础模型，提升特定领域的准确性。
场景：金融风控模型定制、医疗诊断辅助、法律文档分析
同类：OpenAI Fine-tuning, Google Vertex AI Tuning
知识库（Knowledge Bases）
全托管的 RAG（检索增强生成）服务，自动将企业文档转化为向量索引，让 AI 基于企业知识回答问题。
场景：企业内部知识问答、客服机器人、技术文档搜索
同类：LangChain + Pinecone, LlamaIndex
Guardrails（安全护栏）
为 AI 应用添加安全防护，过滤有害内容、防止敏感信息泄露、限制话题范围、检测幻觉。
场景：面向客户的 AI 应用，合规要求严格的金融/医疗场景
同类：OpenAI Moderation API, NVIDIA NeMo Guardrails
模型优化（Optimization）
自动优化模型推理性能，包括量化、蒸馏、推理加速，在保持质量的前提下降低成本和延迟。
场景：高并发 API 服务，对延迟敏感的实时应用
同类：vLLM, TensorRT-LLM
AGENT BUILDERS · 智能体构建框架
Amazon Nova Act 垂直集成
端到端的 AI Agent 模型，专为在浏览器中执行复杂任务设计。能理解网页、点击按钮、填写表单、导航页面，完成多步骤的 Web 操作。
场景：自动化网页操作、RPA 替代、电商价格监控、自动化测试
同类：Anthropic Computer Use, OpenAI Operator
Strands Agents 开源 SDK
灵活的开源 Agent SDK，用简单的 prompt + tools 模式构建 AI Agent。支持多 Agent 协作、工具调用、记忆管理。"给 Agent 一个目标和工具，让 LLM 自己决定怎么做。"
场景：自定义 Agent 开发，多 Agent 协作系统，需要完全控制的场景
同类：LangChain/LangGraph, CrewAI, AutoGen
BEDROCK AGENTCORE · 智能体运行时
AgentCore Runtime
Serverless 的 Agent 运行时，自动扩缩容，按秒计费。每个 Agent session 运行在独立的 MicroVM 中，安全隔离。
场景：生产环境 Agent 部署，需要弹性扩展和安全隔离
同类：Azure AI Agent Service, Google Vertex AI Agent
AgentCore Gateway
统一的 Agent API 网关，提供身份认证、流量控制、负载均衡、协议转换。
场景：多 Agent 系统的统一入口，企业级 API 管理
同类：Kong AI Gateway, Portkey
AgentCore Identity & Policy
Agent 级别的身份认证和策略控制，定义 Agent 可以访问哪些资源、调用哪些工具、执行哪些操作。
场景：企业安全合规，Agent 权限精细化管理
同类：无直接竞品（AWS 独有）
AgentCore Memory
Agent 的长期记忆系统，跨 session 保持上下文，让 Agent 能"记住"之前的交互和学到的知识。
场景：个性化助手，需要长期记忆的客服 Agent
同类：Mem0, Zep
AgentCore Observability
Agent 运行的可观测性，追踪每一步推理、工具调用、决策过程，帮助调试和优化 Agent 行为。
场景：Agent 调试、性能优化、异常检测
同类：LangSmith, Arize Phoenix
AgentCore Evaluations
Agent 质量评估框架，自动化测试 Agent 的准确性、安全性、可靠性，确保上线前达到质量标准。
场景：Agent 上线前的质量门禁，持续质量监控
同类：Ragas, DeepEval
🏗️Infrastructure — 基础设施层
AI 的底座。从自研芯片到模型训练平台，提供构建和运行 AI 所需的一切计算、存储和数据基础设施。
AI COMPUTE · AI 计算
AWS Trainium 自研芯片
AWS 自研 AI 训练芯片，专为大规模模型训练优化。Trainium2 提供比上一代 4 倍的训练性能，支持万亿参数模型训练。
场景：大模型预训练、微调，替代 NVIDIA GPU 降低训练成本 40%+
同类：NVIDIA H100/H200, Google TPU v5
AWS Inferentia
AI 推理专用芯片，为模型推理提供高吞吐、低延迟、低成本的计算。Inferentia2 支持 LLM 推理，性价比领先。
场景：生产环境模型推理部署，实时 API 服务，降低推理成本 50%+
同类：NVIDIA T4/L4, Google TPU Inference
GPU 实例（P5/P4）
基于 NVIDIA GPU 的 EC2 实例，P5 搭载 H100，适合需要 CUDA 生态的 AI 工作负载。
场景：需要 NVIDIA CUDA 生态的训练和推理，HPC 计算
同类：Azure NDm A100, GCP A3 (H100)
AMAZON SAGEMAKER · 模型构建与数据基础
SageMaker — 模型构建
端到端的机器学习平台，涵盖模型构建（Model Building）、训练（Training）、部署（Deployment）全流程。
场景：自定义模型开发，从数据准备到模型上线的完整 ML 工作流
同类：Google Vertex AI, Azure ML Studio
SageMaker HyperPod
大规模分布式训练集群，自动故障恢复，支持数千 GPU/Trainium 芯片的并行训练。
场景：基础模型预训练，需要数百到数千 GPU 的大规模训练任务
同类：Azure AI Infrastructure, GCP ML Hub
SageMaker MLOps
模型生命周期管理，包括模型注册、版本控制、A/B 测试、自动化部署流水线。
场景：企业级 ML 模型的持续集成和持续部署（CI/CD for ML）
同类：MLflow, Kubeflow, Weights & Biases
数据基础设施
涵盖数据处理（Data Processing）、治理（Governance）、存储与数据库（Storage & Databases）、向量搜索（Vector Search）。
场景：RAG 应用的向量数据库、数据湖构建、数据质量治理
同类：Databricks, Snowflake, Pinecone
